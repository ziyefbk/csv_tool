# 更新日志

## [0.2.0] - 2024

### ✨ 新增功能

#### 索引持久化
- ✅ 自动保存索引到 `.csv.idx` 文件
- ✅ 再次打开文件时自动加载索引
- ✅ 索引有效性验证（文件修改时间、大小等）
- ✅ 索引失效时自动重建
- ✅ 性能提升：重复打开文件速度提升20-40倍

**技术细节**:
- 使用 `bincode` 进行二进制序列化
- 索引文件格式：`<csv_file>.csv.idx`
- 索引元数据包含：文件路径、大小、修改时间、版本号

**API变更**:
- 新增 `RowIndex::index_file_path()` - 生成索引文件路径
- 新增 `RowIndex::save_to_file()` - 保存索引到文件
- 新增 `RowIndex::load_from_file()` - 从文件加载索引
- 新增 `RowIndex::is_index_valid()` - 验证索引有效性
- 新增 `IndexMetadata` 结构体

**行为变更**:
- `CsvReader::open()` 现在会自动尝试加载索引
- 如果索引存在且有效，直接使用，否则构建新索引
