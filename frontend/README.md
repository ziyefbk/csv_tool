# CSV Tool Frontend

React + TypeScript + Tailwind CSS 前端应用

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 项目结构

```
frontend/
├── src/
│   ├── components/     # React 组件
│   │   ├── CSVTable.tsx      # CSV 表格组件
│   │   ├── FileInfo.tsx      # 文件信息组件
│   │   └── Pagination.tsx    # 分页组件
│   ├── App.tsx         # 主应用组件
│   ├── main.tsx        # 入口文件
│   └── index.css       # 全局样式
├── package.json
└── vite.config.ts
```
